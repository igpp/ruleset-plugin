Java Application
IGPP Ruleset Processor
The IGPP Ruleset Processor is a Java implementation of the IGPP Ruleset Language.

Contains the class libraries of the ruleset processing engine which can be embedded in other applications and a command-line tool process rulesets and generate documents based on templates.

igpp.ruleset.Labeler
Generate documents based on a ruleset.
