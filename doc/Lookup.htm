PDS/PPI Lookup Plug-in

A plug-in for use with the PDS/PPI ruleset processor to lookup values in a table using different search methods.

Here's what is available:

  • User's guide
  • Lookup.java: Source code
  • Lookup.class: Compiled Java application
  • Lookup$Item.class: Nested class needed for lookup.class
Installation instructions:
  1. Place the "Lookup.class" and "Lookup$Item.class" file in $JAVA_HOME/jre/lib/pdsppi
  2. Add "$JAVA_HOME/jre/lib/pdsppi" to your CLASSPATH environment variable.
Note: The application also uses various classes contained in pdsppi.jar
You will need to download and install this Jar file as well.
