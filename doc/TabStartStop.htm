PDS/PPI TabStartStop Plug-in

A plug-in for use with the PDS/PPI ruleset processor return a portion (column) of the first and last rows in an ASCII table.

Here's what is available:

  • User's guide
  • TabStartStop.java: Source code
  • TabStartStop.class: Compiled Java application
Installation instructions:
  1. Place the "TabStartStop.class" file in $JAVA_HOME/jre/lib/pdsppi
  2. Add "$JAVA_HOME/jre/lib/pdsppi" to your CLASSPATH environment variable.

Note: The application also uses various classes contained in pdsppi.jar
You will need to download and install this Jar file as well.
