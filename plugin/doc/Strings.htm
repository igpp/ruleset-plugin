PDS/PPI Strings Plug-in

A plug-in for use with the PDS/PPI ruleset processor to determine length, change case, index, and subset strings.

Here's what is available:

  • User's guide
  • Strings.java: Source code
  • Strings.class: Compiled Java application
Installation instructions:
  1. Place the "Strings.class" file in $JAVA_HOME/jre/lib/pdsppi
  2. Add "$JAVA_HOME/jre/lib/pdsppi" to your CLASSPATH environment variable.

Note: The application also uses various classes contained in pdsppi.jar
You will need to download and install this Jar file as well.
