PDS/PPI TargetPhrase Plug-in

A plug-in for use with the PDS/PPI ruleset processor to create a properly punctuated phrase describing a list a values.

Here's what is available:

  • User's guide
  • TargetPhrase.java: Source code
  • TargetPhrase.class: Compiled Java application
Installation instructions:
  1. Place the "TargetPhrase.class" file in $JAVA_HOME/jre/lib/pdsppi
  2. Add "$JAVA_HOME/jre/lib/pdsppi" to your CLASSPATH environment variable.

Note: The application also uses various classes contained in pdsppi.jar
You will need to download and install this Jar file as well.
