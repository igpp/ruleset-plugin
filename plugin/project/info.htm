Java Application
IGPP Ruleset Plug-ins
Plug-ins are callable tasks from within a ruleset that is processed by the IGPP Ruleset Processor.

All plug-ins in this collection are written in Java for maximum portability.

Available plug-ins are:

igpp.ruleset.plugin.Compare
igpp.ruleset.plugin.FFHScan
igpp.ruleset.plugin.FormatDescription
igpp.ruleset.plugin.IMath
igpp.ruleset.plugin.Lookup
igpp.ruleset.plugin.SpreadSheet
igpp.ruleset.plugin.Strings
igpp.ruleset.plugin.TabStartStop
igpp.ruleset.plugin.TargetPhase
igpp.ruleset.plugin.Time
